# 何为多组学

## 多组学定义

随着高通量组学平台的发展，生物医学研究大多采取了多组学技术结合的方法，不同组学来源（如遗传学、蛋白质组学和代谢组学）的数据可以通过基于机器学习（Machine Learning，ML）的预测算法进行整合，以揭示系统生物学的复杂工作

### 单细胞问题

#### 单细胞多组学

单细胞多组学技术和方法通过同时整合各种单模态多组学方法，对转录组、基因组、表观基因组、表转录组、蛋白质组、代谢组和其他（新兴）多组学进行剖析，从而描述细胞状态和活动
