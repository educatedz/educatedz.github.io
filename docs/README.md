# 个人博客搭建教程
> An awesome project.

## 简介
hexo、vuepress、jekyll、docsify都能满足你搭建博客的需求，不过docsify是最容易上手的，不会生成静态网页。而且支持中文，主题，离线，搜索，目录，代码高亮，数学公式，图片，视频，音频，音乐。
